## Prettier

```js
npm install --save-dev --save-exact prettier

// 格式化所有文件，npx命令是使用当前项目下的prettier
npx prettier --write .

```

配置文件需要自己手动创建，支持`.prettierrc.JSON；.prettierrc.js；prettier.config.js`
